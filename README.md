# Python Banking System

## Overview
A secure, console-based ATM simulator built with Python. This project allows users to create accounts, manage their savings, and perform transactions with data persistence.

## Features
- **User Authentication:** Secure Login and Signup functionality.
- **Transactions:** Deposit, Withdraw, and Check Balance.
- **Data Persistence:** Automatically saves user data to a local text file.
- **Validation:** Prevents overdrafts and handles invalid inputs.

## Technologies Used
- Python 3
- File Handling (Text files)
